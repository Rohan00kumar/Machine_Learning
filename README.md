## Installation

### Prerequisites
- Python 3.7+
- Pip

### Steps
1. Clone the repository:
    ```sh
    git clone https://github.com/your-username/machine-learning.git
    ```
2. Navigate to the project directory:
    ```sh
    cd machine-learning
    ```
3. Install the dependencies:
    ```sh
    pip install -r requirements.txt
    ```
